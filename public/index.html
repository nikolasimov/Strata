Architecture

Built in layers

Each layer of Strata has a single responsibility. Together they form a complete HTTP server you can trace from socket to response.

layer 01
TCP socket server
Opens a raw TCP socket, binds to a port, and accepts incoming connections. The foundation everything else sits on.
live
layer 02
HTTP parser
Reads raw bytes from the socket and constructs a structured request object — method, path, headers, body.
live
layer 03
Request router
Matches the parsed path against registered routes and dispatches to the correct handler function.
live
layer 04
Static file resolver
Serves files from the /public directory. Detects MIME types for HTML, CSS, JS and more.
live
layer 05
Middleware pipeline
A composable chain of functions that can inspect, modify, or short-circuit every request and response.
planned
layer 06
Reverse proxy
Forwards requests to upstream services — the last and most complex layer to implement.
planned
Current features

What works today

The core HTTP machinery is complete. Strata can serve a real website right now.

⌁
TCP networking
Raw socket handling — no socketserver, just socket.socket.
⟶
HTTP/1.1 parsing
Parses method, path, version, headers, and body from raw byte streams.
◎
Response builder
Constructs well-formed HTTP responses with correct status lines and headers.
⊞
Route matching
Register handlers for paths. Requests are dispatched to the first match.
⊡
Static file serving
Serve any file from /public over HTTP with correct MIME types.
≋
MIME detection
Automatically sets Content-Type for HTML, CSS, JS, and more.
Request lifecycle

A request, step by step

Trace exactly what happens when a browser hits Strata on port 8080.

GET /index.html HTTP/1.1
⌁
TCP connection accepted
socket.accept() → (conn, addr) — client is 127.0.0.1:51240
⟶
Raw bytes received
conn.recv(4096) → b"GET /index.html HTTP/1.1\r\nHost: ..."
◎
HTTP request parsed
method="GET" path="/index.html" version="HTTP/1.1"
⊞
Route matched
No dynamic route — falling through to static file handler
⊡
File resolved
./public/index.html → exists MIME: text/html
✓
Response sent
HTTP/1.1 200 OK Content-Type: text/html Content-Length: 1842
Internals

The core loop

This is the heart of Strata. Everything else is just careful plumbing around these few lines.

while True: conn, addr = server_socket.accept() # block until a client connects raw = conn.recv(4096) # read raw bytes off the wire request = http.parse(raw) # parse into a Request object handler = router.match(request.path) # find the right handler response = handler.handle(request) # generate a Response conn.sendall(response.to_bytes()) # write it back conn.close() # done — HTTP/1.0 style
